# Exodus

This is **exodus coordinator** which acts as the name suggests for units across your network.

---

## Set up

### 1. Compile Binaries

``` bash

make

```

### 2. Quick-Start

``` bash

./exodus-coordinator

```

Just make sure you're running this on a homelab or a dedicated server with a stable internet connection.

---



## Clean up

``` bash

make clean

```
